Ce fichier README a pour but de renseigner sur le projet en c++ intitulé TP_LETHIES_KAMLI. 
Celui-ci a été programmé sous Visual Studio et a pour but la réalisation de filtre sur une image (filtre médian, filtre gaussien, gradient, dilatation, érosion, 
détection de contours ainsi que le seuillage sont réalisables). 
Les bibliothèques d’opencv ont été nécessaires à la réalisation du projet. 
De plus ce projet devait respecter certains critères comme la réalisation d’une structure MVC.

Pour lancer le programme, il vous suffit de lancer la solution sous Visual Studio du projet intitulée LoadImage.sln puis de lancer le débogueur windows local, 
ou de lancer l'exécutable en suivant le chemin: x64/Debug/LoadImage. 

Ce projet a été réalisé par KAMLI Younes et LE THIES Loann.
